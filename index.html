Featured Project

myKart - Hyperlocal Platform

myKart is a full-stack web application that helps users discover nearby local shops and reserve products before visiting. The platform aims to bridge the gap between customers and local businesses by providing a digital marketplace for neighbourhood stores.

The frontend is built using Next.js, React, TypeScript, and Tailwind CSS, while the backend uses Spring Boot, JPA/Hibernate, and H2 Database (Development). Customers can browse shops, view products, manage carts, place reservations, and track orders. Shop owners can register their stores and manage their business through a dedicated dashboard.

The platform includes features such as user authentication, shop listings, product browsing, cart management, order tracking, customer feedback, and location-based shop discovery. It also incorporates SEO optimization using metadata and structured data for improved search engine visibility.

Overall, myKart demonstrates full-stack development skills by integrating a modern frontend, RESTful backend services, database management, and a practical real-world business use case.
